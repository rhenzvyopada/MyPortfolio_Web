# Professional Portfolio Website

A modern, minimalist portfolio website built with plain HTML, CSS, and JavaScript. Features a black, white, and grayscale color palette with elegant animations and smooth scrolling.

## Features

- **Hero Section**: Professional introduction with call-to-action buttons
- **About Me**: Biography, skills with animated progress bars, and experience timeline
- **Projects**: Responsive card layout with hover animations and technology tags
- **Contact**: Working contact form with validation and social media links
- **Footer**: Navigation links, social links, and copyright
- **Back to Top**: Floating button with smooth scroll
- **Fully Responsive**: Mobile-first design with breakpoints at 480px, 768px, and 968px
- **Accessible**: Semantic HTML, ARIA labels, keyboard navigation, and reduced motion support

## Design Features

- Black, white, and grayscale color palette
- Modern minimalist design with strong typography (Inter font)
- Consistent spacing using CSS custom properties
- Elegant fade-in animations on scroll
- Smooth scrolling navigation
- Subtle hover effects on cards, buttons, and links
- Professional appearance throughout

## Technologies Used

- **HTML5**: Semantic markup with proper accessibility attributes
- **CSS3**: Modern CSS with custom properties, Grid, Flexbox, and animations
- **JavaScript (ES6+)**: Vanilla JS for interactivity, no dependencies
- **Google Fonts**: Inter font family for modern typography

## Responsive Breakpoints

- **Desktop**: 968px and above
- **Tablet**: 768px to 967px
- **Mobile**: Below 768px
- **Small Mobile**: Below 480px

## Performance

- Zero dependencies - lightweight and fast
- Optimized scroll events with debouncing
- Intersection Observer for efficient scroll animations
- Passive event listeners for better performance
- Minimal DOM manipulation

## Accessibility

- Semantic HTML5 elements
- ARIA labels for interactive elements
- Keyboard navigation support
- Focus indicators for all interactive elements
- Reduced motion support for users who prefer it
- Proper heading hierarchy
- Alt text for images (when added)

## Getting Started

1. Clone or download the project
2. Open `index.html` in any modern web browser
3. No build process or dependencies required!

## Customization Guide

### Personal Information

1. **Name**: Replace "Your Name" throughout the HTML
2. **Contact Info**: Update email, location, and social media links
3. **Bio**: Edit the biography text in the About section
4. **Skills**: Modify skill names and progress percentages
5. **Experience**: Update timeline with your work history
6. **Projects**: Add your own projects with descriptions and technologies

### Styling

1. **Colors**: Modify CSS custom properties in `:root` (css/styles.css)
2. **Fonts**: Change the Google Fonts link and font-family variable
3. **Spacing**: Adjust spacing variables for consistent layout
4. **Animations**: Modify transition and animation timings

### Images

Replace SVG placeholders with actual images:
- Hero section portrait
- About section profile photo
- Project screenshots
